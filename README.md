# Chen Si's Academic Homepage

Based on [Jon Barron](https://jonbarron.info/)'s website template, with layout inspiration from [Yulin Liu](https://liuyulinn.github.io/).

## How to Edit

1. **Bio**: Edit the header section in `index.html` — update your description, advisor, email, and links.
2. **Photo**: Replace the placeholder with your photo — save as `images/profile.jpg` and uncomment the `<img>` tag.
3. **News**: Add/remove `<li class="news-item">` entries in the News section.
4. **Publications**: Copy a `<tr class="pub-row">` block and fill in your paper details. Set `data-selected="true"` for key papers.
5. **Experience**: Update the Experience section with your institutions and roles.
6. **Honors/Services**: Fill in your awards and reviewer info.
7. **CV**: Place your CV PDF at `data/ChenSi-CV.pdf`.

## Deploy on GitHub Pages

1. Create a repo named `<your-username>.github.io`
2. Push all files to the `main` branch
3. Your site will be live at `https://<your-username>.github.io/`
